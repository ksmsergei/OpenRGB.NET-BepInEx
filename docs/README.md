# OpenRGB.NET for BepInEx — Documentation

Welcome to the full documentation for **OpenRGB.NET for BepInEx**, a C# library that provides an easy way to control RGB lighting via [OpenRGB](https://openrgb.org/) from BepInEx plugins (e.g. for Lethal Company mods).

## Contents

| [API Reference](api-reference.md) | Full reference for all classes, methods, and types |
| [Examples](examples.md) | Practical code examples and common usage patterns |

## Prerequisites

- [OpenRGB](https://openrgb.org/) running with the SDK server enabled (default port: `6742`)
- BepInEx installed in the game directory
- .NET / C# BepInEx plugin project
